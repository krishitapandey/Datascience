Movie Success Prediction & EDA Pipeline
This project implements a robust data science workflow to analyze movie metadata, clean complex datasets, perform feature engineering, and train machine learning models to predict high-quality movies.

1 . Load a csv dataset and perform basic eda (head , info , describe)
2. Handle missing values using multiple strategies
3. Detect and remove duplicate rows
4. Perform Label Encoding and One Hot Encoding
5. Create visualization (bar , line , scatter)
6. Identify and treat outliers using boxplot
7. Perform feature enginnering (create new columns)
8. Apply normalization and standardization
9. Split dataset into train and test 
10. Train a Logistic Regression model and evaluate
11. Implement Decision Tree and compare result 
12. Apply K-Means clustering and use Elbow method
13. Build a simple API using FastAPI to serve predictions
14. Scrape data using Requests + BeautifulSoup and export CSV
15. Automate brosert task using Selenium (login/data extraction)
